# 🔷 Release Notes v1.2.1

**Release Date:** 2026-04-21  
**Version:** 1.2.1

## 🚀 Overview

Ta wersja stabilizuje mechanizmy zarządzania kontekstem oraz automatyzuje telemetrię HUD, eliminując "amnezję architektoniczną" i śmiecenie plikami tymczasowymi.

## 🛠️ Key Improvements

### 1. Telemetry Automation (Zero-Config HUD)
- Wprowadzono dekorator `@telemetry_gate` w `server.py`, który automatycznie wywołuje `start_request`.
- HUD teraz resetuje bufory i liczniki tokenów przy każdym nowym zadaniu bez konieczności manualnej interwencji agenta.

### 2. File Leak Fix (Context Engine)
- Rozwiązano problem z wyciekiem uchwytów plików w `ContextBuilderEngine._atomic_write`.
- Usunięto redundantne tworzenie plików `.tmp`, które zaśmiecały katalog `.context/`.

### 3. SOS Sync Stabilization (Backlog Fixes)
- Poprawiono błąd regex w `qwen_update_task`, który uniemożliwiał poprawne odznaczanie zadań w `BACKLOG.md` przy użyciu standardowych checkboxów.
- Zastąpiono hardcoded ścieżki dynamiczną rezolucją przez `DEFAULT_SOS_PATHS`, co poprawia przenośność serwera między różnymi workspace'ami.

### 4. Persistent Context Chain
- Pełne wsparcie dla `depends_on_adr` i wstrzykiwania blueprintów architektonicznych do podzadań.
- Wdrożono **Tiered Discovery** dla `workspace_root`, co zapobiega crashom serwera w trybie pracy na wielu katalogach.

## 📦 Dependencies
- Bez zmian w stosunku do v1.2.0.
